# Remind Me

## Description

This is a simple reminder app that allows users to create reminders and set a time for them to be reminded. The user can also edit and delete reminders. The app uses a postgres database to store the reminders.

## Built with

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Clerk Auth](https://clerk.dev/)
- [Vercel Postgres](https://vercel.com/docs/solutions/postgres)
- [Prisma](https://www.prisma.io/)
- [ShadcnUI](https://ui.shadcn.com/)
